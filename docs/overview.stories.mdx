# Search UI Components

React components for search and discovery.

These components work alongside [@osuresearch/service-graph](https://github.com/osuresearch/service-graph) search services to provide an easy to setup end-to-end search experience.

## Installation

Install using your package manager of choice:

- `yarn add @osuresearch/search-ui`
- `npm i @osuresearch/search-ui`
- `pnpm i @osuresearch/search-ui`

## IMDb Demo

This demo uses the typical IMDb media dataset for searching and filtering through movies, games, and tv series.
